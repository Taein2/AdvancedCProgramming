# Advanced_C
###1. 프로젝트 개요
##1-1. 프로젝트 목적
학생들이 복수전공을 신청하고 취소하기 위한 서비스를 기획한다.  

##1-2. 프로젝트 기능
학생들에 정보에서 해당하는 학생의 복수 전공 신청과 취소를 할 수 있는 기능이 있다. 신청과 취소한 내용을 확인 할 수 있으며, 전공 개수를 가진 학생들의 목록을 볼 수 있다. 전체 학생들의 리스트 또한 볼 수 있다.
